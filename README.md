# DEV.HJ — Backend Developer Portfolio

GitHub Pages 배포용 1페이지 포트폴리오 웹사이트.

## 구조
```
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── images/
```

## 커스터마이징
- `index.html` 에서 `[NAME]`, `[EMAIL 입력하세요]`, `[GITHUB_URL 입력하세요]` 를 실제 정보로 교체하세요.
- GitHub Pages: 저장소 Settings → Pages → Branch: main, folder: / (root)
